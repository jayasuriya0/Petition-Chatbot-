🔔 Notification System Test Console

1. Check Authentication Status

Verify that you're logged in as a department before testing.

2. Fetch Notifications

Retrieve all notifications for the logged-in department.

3. Mark Notifications as Read

Test marking notifications as read (requires notification ID).

Note: To mark a specific notification as read, first fetch notifications above to see IDs.

4. Trigger Daily Report

Manually trigger a daily summary report email.

Note: This will send an email to the department's email address. Check your inbox!

5. Trigger Weekly Report

Manually trigger a weekly performance report email.

Note: This will send an email to the department's email address. Check your inbox!

6. Test High Urgency Alert

To test high urgency email alerts:

  1. Go to the petition submission page
  2. Fill out the form
  3. Set Urgency to "High"
  4. Submit the petition
  5. Check the department email inbox for the alert
  6. Check the bell icon in department dashboard for notification

7. View Department Dashboard

Open the department dashboard to see the bell icon notification system in action.
